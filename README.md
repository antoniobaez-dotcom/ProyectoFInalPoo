# Descripción
Este proyecto se diseño teniendo en cuenta que se buscaba aplicar el programa para "solucionar" un posible problema real dentro de la rama de la ingenieria mecatrónica.
El proyecto consta de la simulacion(En base a datos ingresados por el usuario) de la temperatura dentro de un horno térimco. 
En el proyecto se utilizan todos los conocimientos adquiridos a lo largo de todo el curso como pueden ser: El uso de clases abstractas,
métodos abstractos, herencia, extenciones e interfaces gráficas

### Instrucciones de Ejecución desde IDE
1. Abrir NetBeans (o el IDE de preferencia).
2. Seleccionar `File` -> `Open Project`.
3. Buscar y seleccionar la carpeta `ProyectoPOO` que se encuentra en la raíz de este repositorio.
4. Hacer clic en `Run Project`.
